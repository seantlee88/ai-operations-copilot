# AI Operations Copilot

Live App:
https://ai-operations-copilot-sean.streamlit.app/

## Overview
AI Operations Copilot is a Streamlit-based AI assistant that helps summarize business documents, extract action items, timelines, and costs, and answer questions from uploaded documents such as estimates, reports, and employee handbooks.

This project is designed for operations, construction, restoration, and service companies that deal with large amounts of documentation and job estimates.

## Features
- Upload TXT and PDF documents
- Document preview
- AI-generated summary
- Extract action items
- Extract costs and pricing details
- Ask questions about the document
- Built for real-world estimate and operations documents

## Tools Used
- Python
- Streamlit
- OpenAI API
- PyPDF
- Pandas

## Example Use Cases
- Construction estimates
- Equipment rental quotes
- Employee handbooks
- Job reports
- Insurance scope documents
- Project timelines

## How to Run Locally
pip install -r requirements.txt
streamlit run app.py
